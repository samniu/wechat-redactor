# 微信聊天截图打码工具

这是一个纯前端的在线工具，用于给微信聊天截图打码。工具允许你上传或粘贴聊天截图，通过矩形或画笔工具对特定区域进行像素化、模糊或黑框处理，并可撤销/重做、导出图片、保存/载入遮罩 JSON。

## 功能特性

* **上传/粘贴图片**：支持拖拽上传和剪贴板粘贴图片。
* **工具选择**：矩形和画笔两种选择方式。
* **打码效果**：像素化、模糊和黑框，可调节强度或黑框圆角与颜色。
* **撤销/重做**：任意撤销和恢复操作。
* **遮罩保存/载入**：将已应用的遮罩保存为 JSON，再次载入以复用遮罩。
* **导出图片**：支持导出 PNG 或 JPG 格式。

## 使用方法

1. 打开 `index.html`。
2. 通过“选择文件”上传图片，或直接粘贴截图。
3. 选择 **矩形**或**画笔**工具并选择打码效果（像素化、模糊或黑框）。
4. 在图片上拖拽或涂抹需要打码的区域，调整强度即可实时预览效果。
5. 如有需要可撤销/重做操作，或保存/载入遮罩配置。
6. 完成后点击“导出 PNG”或“导出 JPG”保存打码后的图片。

## 部署

本项目为纯静态页面，无需后端，支持直接部署到 GitHub Pages、Netlify 或任何静态文件服务器。

### GitHub Pages

1. 创建新的 GitHub 仓库，将本目录内容推送至 `main` 分支。
2. 通过仓库 Settings → Pages，设置 **Build and deployment** 为 GitHub Actions。工作流脚本在 `.github/workflows/deploy-pages.yml` 中已提供。
3. 推送代码后，GitHub Actions 会自动将静态文件部署到 Pages，几分钟后即可访问。

### Netlify

可以直接使用 [Netlify Drop](https://app.netlify.com/drop) ，将整个目录拖入即可实现自动部署，并获取一个可访问的网址。
